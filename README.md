# MERN CodeSync - Collaborative Code Editor

## Installation
### Prerequisites
```
Node.js
Git
```

### Setup
```
git@github.com:Abhishekrai007/realtime-collaborative-codeeditor.git
cd shop-video-platform

cd api
npm i

cd ..

cd client
npm i

```
### Run
```
cd api
npm start

cd client
npm run dev

```
